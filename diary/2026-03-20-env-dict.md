# 2026-03-20 修正と進捗

## 修正1: environment_variables dict 統一

### 問題
- ワーカープロセスが `os.environ` に直接書き込むのはアンチパターン
- `TaskContext` に GPU ID などの環境情報を個別キーで保有していた

### 修正内容

1. **スケジューラ側 (resource_scheduler.py)**
   - `next_case()` でいくつかの environment variables をまとめて dict 化
   - `context["environment_variables"]` に以下をセット:
     ```python
     {
         "CUDA_VISIBLE_DEVICES": "0",
         "NVIDIA_VISIBLE_DEVICES": "0",
         "gpu_ids": "0",
         "gpu_id": "0",
         "XLA_PYTHON_CLIENT_PREALLOCATE": "false"  # 必要に応じて
     }
     ```

2. **ワーカー側 (run_smolyak_experiment_simple.py)**
   - `_setup_gpu_environment()` を修正
   - `context["environment_variables"]` dict を読み取り
   - 各 key-value を `os.environ` に設定
   - JAX インポート前に設定完了

## 修正2: Worker プロトコルの型定義統一

### 問題  
- `Worker` プロトコルが3つの型パラメータを使用していた
- プロトコル定義では `Worker[T, U]` のみが正しい

### 修正内容

**protocols.py**:
```python
class Worker(Protocol[T, U]):
    """ワーカーはタスク（case）を受け取り `TaskContext` 下で実行"""
    task: Callable[[T], U]
    def __call__(self, case: T, context: TaskContext) -> int: ...
    def resource_estimate(self, case: T) -> ResourceEstimate: ...
```

**resource_scheduler.py**:
```python
class FullResourceWorker(Worker[T, int], Protocol[T]):
    """GPU リソース推定機能を持つワーカー"""
    def resource_estimate(self, case: T) -> FullResourceEstimate: ...
```

**run_smolyak_experiment_simple.py**:
```python
class SmolyakWorker(Worker[dict[str, Any], int]):
    """Smolyak 積分器を実行するワーカー"""
    # 型パラメータ:
    # - T = dict[str, Any] （ケース）
    # - U = int （終了コード）
```
