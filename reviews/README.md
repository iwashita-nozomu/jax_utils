# reviews

`reviews/` は、コードレビュー報告とレビュー後の進捗整理を置くディレクトリです。
レビュー成果物はここへ集約し、Git 管理します。

## 現在の文書

- `CODE_REVIEW_SUMMARY.md`
  - レビュー全体の要約です。
- `CODE_REVIEW_REPORT.md`
  - モジュールごとの詳細なレビュー報告です。
- `DETAILED_CODE_REVIEW.md`
  - 個別指摘と改善提案を中心にしたレビュー文書です。
- `IMPLEMENTATION_PROGRESS_REPORT.md`
  - レビュー後の実装改善進捗をまとめた文書です。

## 状態ラベル

- `Verified`
  - 現在のリポジトリ状態と整合している確認済み文書です。
- `Working Note`
  - 作業途中の所見や provisional な指摘を含みます。
- `Historical Artifact`
  - 過去時点のレビュー成果物をそのまま保存した文書です。
